# 💰 Expense Tracker

A simple command-line Expense Tracker built using Python.

This project was created to practice Python fundamentals, Object-Oriented Programming (OOP), file handling, and JSON data storage.

---

## 📌 Features

- ➕ Add a new expense
- 📄 View all expenses
- 🔍 Search expenses by category or description
- ✏️ Update an existing expense
- 🗑️ Delete an expense
- 📊 View monthly expense summary
- 💾 Save data permanently using a JSON file

```bash

```

2. Navigate to the project folder

```bash

```

3. Run the application

